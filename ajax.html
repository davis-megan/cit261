Requesting a JSON file through AJAX

AJAX Basics

What is AJAX?

  • Stands for Asynchronous JavaScript And XML
  • Not a programming language
  • Uses JavaScript and HTML DOM to display/use data
  • Also uses a browser built in object to request data from a web server
  • File it's requesting must be on the same server
  • Although the "X" stands for XML, JSON is more frequently used because it's more convenient

Why use it?

  • Use it to update parts of a webpage without reloading the whole thing

How to use AJAX
